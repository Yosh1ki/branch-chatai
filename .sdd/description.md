# 実装したい機能

## 概要
chats/[chatId]のprompt入力の機能の修正
## 詳細

### promptカードの修正
- promptカードにはchats画面のpromptカードと同じように矢印のマークをつける
- 文が何行にもなる場合は長さに応じてテキストボックスを広げる、しかし一気に広げるとかはしないでほしい

### promptが送信された時の挙動
- 送信されたら送信した文を元に、テキストボックスからUser Bubbleに変化させる。user bubbleから伸びる線も表示させる
- AIのレスポンスを表示する
- 表示するカードは使い回しで表示する内容だけを変える
- AIレスポンスからまた線を伸ばしテキストボックスを表示させる
- 線の両隣には新しいブランチピルを表示させる
