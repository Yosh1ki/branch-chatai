# 要件定義書

## 機能概要
chats画面に表示されるチャットカードを、新しい順/古い順で並べ替えできるようにする。chatカード一覧の上部にセレクトボタンを設置し、「最近のチャット」の文言の右側に配置する。

## ユーザーストーリー
- ユーザーとして、chats画面のチャットを新しい順または古い順に並べ替えたい。なぜなら目的のチャットに素早く辿り着きたいから。

## 機能要件
### 要件1：並び順の選択
- chats画面のチャットカード一覧上部にセレクトボタンを配置し、「新しい順」「古い順」を選択できること。
- 受入基準：
  - [ ] 「最近のチャット」の文言の右側にセレクトボタンが表示されること
  - [ ] 「新しい順」と「古い順」を選択できること

### 要件2：表示順の切り替え
- セレクトボタンの選択に応じて、チャットカードの表示順が切り替わること。
- 受入基準：
  - [ ] 「新しい順」を選ぶと最新のチャットが上に表示されること
  - [ ] 「古い順」を選ぶと最古のチャットが上に表示されること

## 非機能要件
- パフォーマンス：切り替え操作が体感的に遅延しないこと
- セキュリティ：既存の認証・権限に従い、ユーザー自身のチャットのみを対象にすること
